# Smart Task – Student Task Tracker

iOS app (iPhone 14, portrait) for tracking academic and personal tasks, built with SwiftUI and Core Data.

## Features

- **Task list** – Pending and completed counts in the header
- **Color-coded rows:**
  - Red – Overdue
  - Yellow – Due within 24 hours
  - Green – Completed
  - Gray – Not urgent / no due date
- **Task types** – Assignment, Quiz, Exam, Personal
- **Local notifications** – Reminders at task due dates
- **Persistence** – Core Data storage
